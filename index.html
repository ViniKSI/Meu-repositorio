Reviews: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir links dos videos de review do episódio 1 ao 58 do Novo Anime Pokémon

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por divisóes da primeira temporada e dos seus rescpectivos videos, após vem outro cabeçalho com a segunda temporada e seus respectivos videos. tela.

O CSS personaliza o visual do site, usando variáveis para cores, definiçoes de espaços entre os cabeçalhos e conteúdos, O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração dos videos.
